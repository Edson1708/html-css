Codigo fuente

Para inserir codigo fuente de cualquier lenguaje de programacion se usa la tag code por ejemplo: System.out.println("hola mundo");

Y para que mantenga el formato como salto de lineas y espacios se usa la tag pre junto con code por ejemplo: 

            
int a = 5;
int b= 8;
int c= a + b;
System.out.println("la suma es "+c);
